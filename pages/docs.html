AngularJS Web Application

  • The NavBar is a directive with its own Controller
  • The Search Result grid is also a directive. However it does not have its own Controller
  • User Authentication is carried out via the Auth0 service. It is free for 7000 users per month!
  • Source Code on GitHub

NodeJS Web Server

  • NodeJS is running using Express to receive Client Requets
  • Data for TV Shows is searched on TVDB using their JSON API
  • TVDB uses JSON Token to authenticate the Server. Token is valid for 24hrs
  • If you do pull the source from GitHub, you will need to create two files under the modules folder:
    • secrets.json : This is the api key for TVDB. You have to get your own API Key and account. The format is:
      {
      "apikey" :"",
      "username" :"",
      "userkey" :""
      }
    • token.json : The token from TVDB is currently saved in this file. The format is:
      {
      "expireTime":"",
      "token":""
      }
